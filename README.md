## Características

<div style="text-align:left">

* Lectura local de contenido.
* Un lector configurable con múltiples visores, direcciones de lectura y otras configuraciones.
* Soporte para rastreadores: [MyAnimeList](https://myanimelist.net/), [AniList](https://anilist.co/), [Kitsu](https://kitsu.app/), [MangaUpdates](https://mangaupdates.com), [Shikimori](https://shikimori.one), y [Bangumi](https://bgm.tv/) soporte.
* Categorías para organizar tu biblioteca.
* Temas claro y oscuro.
* Actualización de horarios de tu biblioteca para nuevos capítulos.
* Crear copias de seguridad localmente para leer sin conexión o en tu servicio de nube deseado.
* Y mucho más...

</div>
